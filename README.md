## Setup/Modifying
1. Clone the repository:
```bash
git clone [Repository URL]
```
2. Navigate to the project directory:
```bash
cd folder
```
3. Install the dependencies (a bit buggy but gets the job done):
```bash
npm install --force
```
4. Run the project in development mode:
```bash
npm run dev
```
